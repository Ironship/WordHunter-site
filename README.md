# Word Hunter website

Official website and privacy policy for [Word Hunter](https://github.com/Ironship/WordHunter).

The download page currently tracks the stable
[Word Hunter 1.0.6 release](https://github.com/Ironship/WordHunter/releases/tag/WordHunter1.0.6),
including direct Windows, Android, and Flatpak downloads plus the pending
[WinGet](https://github.com/microsoft/winget-pkgs/pull/401481) and
[F-Droid](https://gitlab.com/fdroid/rfp/-/work_items/4109) submissions.

The site is deployed to GitHub Pages from the `main` branch by the `Deploy GitHub Pages` workflow.

## Local preview

Serve this directory with any static HTTP server, for example:

```sh
python -m http.server 8000
```

Then open <http://localhost:8000>.
